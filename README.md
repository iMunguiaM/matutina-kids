# Campamento Matutina

Aplicación React/Vite para una experiencia infantil de matutina bíblica con estilo de campamento, mapa de aventura, lectura por voz e insignias.

## Requisitos

- Node.js 20 o superior recomendado
- npm

## Instalar dependencias

```bash
npm install
```

## Ejecutar en modo desarrollo

```bash
npm run dev
```

Luego abre la URL que muestre la terminal, normalmente:

```txt
http://localhost:5173
```

## Crear versión para producción

```bash
npm run build
```

## Vista previa de producción

```bash
npm run preview
```
